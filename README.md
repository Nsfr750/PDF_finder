## ğŸ“¦ Installation

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Quick Install

1. Clone the repository:
   ```bash
   git clone https://github.com/Nsfr750/PDF_finder.git
   cd PDF_finder
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```
